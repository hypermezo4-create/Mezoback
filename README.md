# MEZOBack Starter

Clean-room Android starter for a DNA-style ROM workspace app using the MoveOS visual identity.

## Included now
- MoveOS-inspired dark neon blue/purple theme
- Bottom navigation (Dashboard, Jobs, Logs, Settings)
- DNA-style cards and quick actions
- Starter build pipeline model with live status simulation
- Safe architecture for:
  - project/workspace management
  - unpack/repack jobs
  - super/partition workflow orchestration
  - live reports and logs

## Intentionally not implemented
This starter does **not** include any feature whose main purpose is bypassing device/app security checks.

## Suggested next build steps
1. Connect the script runner to your owned shell tools.
2. Add real unpack/repack command execution with structured logs.
3. Add checkpoints / rollback / temp cleanup.
4. Add patch recipes for safe ROM customization tasks.
5. Add project import/export and result verification.
